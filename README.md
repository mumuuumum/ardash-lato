This is a 2020 remake of the Game Alto's Adventure (2015), which was a remake of Ski Safari (2012).

'Lato' is the name of the main charater and describes someone who is always 'late' and that's why he has to hurry.

Unfortunatelly none of the previous endless runner games was FLOSS. There was TredGamerZ/Legendary (2017), it was open source but still based on Unity. Unity is not open source. The corona SDK engine has been made open source in Jan 2019 and they got a show case example called Endless Sk8boarder, whcih looks like crap and doesn't do good marketing for the engine.

The only suitable way aroudn that was to use a well established and well supported FLOSS engine (LibGDX) to make a complete new endless runner game. Unfortunatelly many problems are also not resolved yet in LibGDX and yet have to be pioneered. That's what this project attempts. Bring a showcase app to the FLOSS community that provides an example implementation for:
* endless procedural random terrain
* 2D and 3D mixture
* realistic waether and day night cycles
* physics without physics engnie
* chained shaders

in LIBGDX.

If you are a game maker too, please feel free to copy everyhing you need from this project. If you are a gamer, please enjoy this game for free, no ads, no optional payments, pure fun.
